# localeCompare(compareString, localesopt, optionsopt) → {number}

The method returns a number indicating whether a reference string comes before or after or is the same as the given string in sort order.

Parameters:
Name Type Attributes Default Description
compareString string | Utf

The string or Utf16 against which the referring Utf is compared.
locales string <optional>

A string with a BCP 47 language tag, or an array of such strings.
options object <optional>

An object with some or all of the following properties:
localeMatcher string <optional>
 "best fit"

Possible values: "lookup" or "best fit"
usage string <optional>
 "sort"

Possible values: "sort" and "search"
sensitivity string <optional>
 "variant"

Possible values: "variant", "case", "accent", and "base"
ignorePunctuation boolean <optional>
 false

Should ignore punctation or not.
numeric boolean <optional>
 false

Whether numeric collation should be used, such that "1" < "2" < "10".
caseFirst string <optional>
 "false"

Whether upper case or lower case should sort first. Possible values are "upper", "lower", or "false" (use the locale's default).
See:
  • https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/localeCompare
